-- 필요한 패키지
1. jsp랑 데이터 베이스 연결할 컨트롤러
2. DB랑 통신할때 데이터 넣을 객체 => 도메인
3. 컨트롤러랑 DB를 연결할 서비스, DAO(repository) 패키지
4. DB에 명령어를 전달할 Mapper 패키지
5. DB에 데이터를 보낼 때 디비 테이블 구조와 매핑시키기 위한 orm 패키지
6. 핸들러 패키지

-- 필요한 폴더
board => 글쓰기, 글 목록 리스트, 글 수정, 세부 글
	register.jsp
user
commnet

--글쓰기 버튼 만들기
1. index.jsp에 버튼 만들기 => 컨트롤러로 이동
2. 컨트롤러에서 register 페이지로 이동 후 글 입력
3. 입력 데이터를 다시 컨트롤러의 insert로 보내기
4. insert에서 받은 데이터 서비스 => DAO => DB 순서로 보내기(POST)

--리스트 버튼
1. index.jsp에 버튼 생성 => 컨트롤러로 이동
2. 컨트롤러에서 DB에 있는 board 데이터 리스트로 가져오기
3. 데이터 리스트를 list.jsp에 뿌려주기

--디테일 버튼
1. 리스트 게시글 마다 a태그 추가 => 컨트롤러로 이동 / 게시글 번호 같이 전달
2. 컨롤러에서 게시글 번호 가지고 해당하는 데이터 가져오기
3. 디테일 페이지 만들 후에 데이터 뿌리기

--수정 삭제 버튼
1. 디테일에서 수정, 삭제 버튼 생성
2. 버튼 클릭 시 컨트롤러로 이동
